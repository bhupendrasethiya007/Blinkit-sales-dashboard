# 🛒 Blinkit Sales Analysis Dashboard

## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of Blinkit's sales performance, customer ratings, product categories, and outlet performance. The dashboard helps stakeholders monitor key business metrics and identify opportunities for growth and operational improvement.

---

## 🎯 Business Objective

To analyze Blinkit's sales data and generate actionable insights related to:

* Total Sales Performance
* Average Sales per Product
* Customer Ratings
* Item Distribution
* Outlet Performance
* Product Category Analysis

---

## 📊 Key KPIs

* Total Sales
* Average Sales
* Number of Items
* Average Rating

---

## 📈 Dashboard Insights

* Sales performance across different outlet types
* Product category-wise sales analysis
* Outlet size and location performance
* Customer rating distribution
* Item type contribution to total sales
* Sales trends and performance comparison

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization
